fishogrow: Back-calculation and growth-curves for fishes in a Bayesian framework
================================================================================

The `fishogrow` package provides a tool to conduct back-calculation
based on otolith readings and fit the Von Bertalanffy growth curve, both
in a Bayesian framework.

Installing and loading fishogrow
================================

`fishogrow` uses Markov Chain Monte Carlo simulations provided by
[stan](https://github.com/stan-dev/rstan/wiki/RStan-Getting-Started).
Therefore, the first step is to install
[stan](https://github.com/stan-dev/rstan/wiki/RStan-Getting-Started).

### GitHub

The best way to install the latest version of `fishogrow` is to install
it from GitHub.

``` r
install.packages("devtools")
devtools::install_github("nschiett/fishogrow", dependencies=TRUE)
library(fishflux)
```

### CRAN

`fishogrow` will be available on CRAN in the future:

``` r
install.packages("fishogrow")
library(fishogrow)
```
